## Product invariants preserved

- Free registration.
- Child expiry bounded by `min(now + 30 days, parent effective expiry)`.
- No child grace period.
- Child owner never receives `CAN_EXTEND_EXPIRY`.
- First-degree label semantics (`[a-z0-9]{8,63}` single label only).

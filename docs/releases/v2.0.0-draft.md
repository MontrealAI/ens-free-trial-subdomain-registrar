# v2.0.0 (Draft) — Identity Primitive Upgrade

> Status: draft release notes; deployment of `FreeTrialSubdomainRegistrarIdentity` is still pending.

## Highlights

- Introduces `FreeTrialSubdomainRegistrarIdentity` as the new flagship contract.
- Preserves free-trial ENS registrar invariants while adding identity token semantics.
- Adds atomic wrapped subname registration + soulbound identity mint.
- Adds permissionless `syncIdentity` and safe `claimIdentity` backfill flow.

## Deployment status honesty

- `FreeTrialSubdomainRegistrar` (`v1.0.0`) remains the only currently live mainnet deployment.
- No new mainnet identity deployment is claimed in this release draft.

## Why major version bump (v2.0.0)

This is a substantial architecture evolution from registrar-only infrastructure to a registrar + identity primitive model with a new contract surface and operator flow, which justifies a major semver increment.
